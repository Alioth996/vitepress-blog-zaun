## Detail

This is a blog theme used vitepress to build. You could [click here](https://visionary-sunflower-dc7ae3.netlify.app/) to see what it looks like.

## Highlights

- [x] pagination
- [x] dark mode
- [x] archives
- [x] tags
- [x] content level
- [x] comments
- [x] lastUpdated
- [x] github actions
- [x] mobile + desktop
- [x] RSS

## Start

```shell
git clone
pnpm install
pnpm run build
pnpm run serve
```
